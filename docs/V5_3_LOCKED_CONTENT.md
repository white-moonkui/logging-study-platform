### 3.2 右侧案例展示区 - 测井曲线异常案例（锁定）

**案例标题**：

```
[成功] 复杂碳酸盐岩储层曲线异常识别与重新测量方案
```

**案例结构**：

```
1. 案例概述
   └─ 碳酸盐岩储层测井曲线异常波动，常规解释方法无法准确识别

2. 测井曲线特征
   ├─ 自然电位(SP)：基线偏移，异常负异常
   ├─ 电阻率(RT)：中高阻背景下的低阻异常段
   ├─ 声波时差(AC)：异常跳变，数值偏高
   └─ 密度(DEN)：与中子曲线交叉异常

3. 问题分析
   ├─ 泥浆侵入影响导致电阻率失真
   ├─ 井眼垮塌造成声波时差异常
   └─ 储层非均质性强，常规测井系列响应复杂

4. 解决方案（成功案例）
   ├─ 采用阵列侧向测井(ALS)校正泥浆侵入影响
   ├─ 调整测速至200m/h进行重复测量
   ├─ 补充核磁共振测井(NMR)验证流体性质
   └─ 综合地质录井资料进行多层对比

5. 最终结果
   ├─ 准确识别出3套高产气层
   └─ 修正后测井解释符合率达92%

6. 经验教训
   └─ 复杂碳酸盐岩储层测井需要综合多种测井资料
```
