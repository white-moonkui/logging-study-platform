### 4. 功能模块结构调整

| 模块             | 子模块                                               | 说明        |
| ---------------- | ---------------------------------------------------- | ----------- |
| 测井知识学习培训 | 测井基础理论、测井仪器设备、测井专业标准、跨专业知识 | 4个二级模块 |
| 典型案例学习     | 交互式学习、典型案例浏览                             | 2个二级模块 |
| 考试系统         | 章节测试、综合测试                                   | 2个模式     |
| 培训效果评估     | 能力雷达图、学习历程、薄弱点分析、进度统计           | 4个卡片     |
| 智能处置推荐     | 对话框 + 文档上传                                    | ChatGPT风格 |
| 知识库管理       | CRUD + 审核流程 + 批量导入                           | 6个分类     |
| 用户管理         | 用户CRUD + 角色权限                                  | 3种角色     |
| AI配置           | 内网/外网大模型配置                                  | API配置     |

### 5. 考试系统详细设计

- **章节测试**：可选择测试范围（8个章节选项）
- **综合测试**：
    - 题库抽题：60%题目（基础难度）
    - AI生成题目：40%（综合难度）
    - 需有交互过程（开始测试 → 答题 → 交卷 → 出分）

### 6. 知识库管理详细设计

- **6个分类**：
    1. 测井基础知识
    2. 测井仪器设备知识
    3. 测井专业标准
    4. 跨专业知识
    5. 测井论文
    6. 测井典型案例

- **批量导入功能**：
    - 导入已审核内容：直接入库
    - 导入新增内容：进入审核流程，审核通过后正式发布
