# 交互式学习模块说明

## 概述

交互式学习模块是测井培训系统的核心功能之一，通过分步骤引导式学习方式，让学员在解决实际问题的过程中掌握测井解释技能。

## 功能特点

### 1. 分步骤学习流程

- 每个案例包含多个学习步骤（通常 3-8 步）
- 每步都有明确的学习目标和任务
- 渐进式难度设计，由浅入深

### 2. 智能提示系统

- 每个步骤提供 2-4 个渐进式提示
- 学员可根据需要获取提示，降低学习难度
- 不影响最终评分，鼓励自主思考

### 3. 即时反馈机制

- 学员提交答案后立即获得反馈
- 正确答案会显示详细解释
- 错误答案引导重新思考

### 4. 进度跟踪

- 实时显示学习进度百分比
- 记录每步完成状态
- 支持断点续学

## 案例示例

### 示例1：复杂碳酸盐岩储层测井识别与分析

**难度**: 中级  
**步骤**: 6步  
**内容**:

1. 岩性识别（利用Pe值判断石灰岩）
2. 储层识别（电阻率、声波时差特征）
3. 孔隙类型判断（成像测井识别裂缝）
4. 孔隙度计算（威利时间平均公式）
5. 流体性质判断（Archie公式计算含水饱和度）
6. 综合解释结论

### 示例2：砂泥岩剖面水淹层识别

**难度**: 中级  
**步骤**: 3步  
**内容**:

1. 水淹现象识别（自然电位幅度变化）
2. 电阻率特征分析
3. 水淹程度判断

### 示例3：页岩气储层测井评价

**难度**: 高级  
**步骤**: 2步  
**内容**:

1. TOC含量计算（ΔlgR法）
2. 脆性指数评价

## API接口

### 1. 开始互动学习

```
POST /api/cases/:id/interactive
Content-Type: application/json

{
    "step": 0  // 从第0步开始
}

Response:
{
    "step": {
        "stepNumber": 1,
        "instruction": "...",
        "hints": [...]
    },
    "totalSteps": 6,
    "currentStepNumber": 0,
    "progress": 0
}
```

### 2. 提交答案

```
POST /api/cases/:id/interactive/answer
Content-Type: application/json

{
    "step": 0,
    "answer": "石灰岩"
}

Response:
{
    "isCorrect": true,
    "feedback": "正确！Pe值约5.0是石灰岩的典型特征...",
    "nextStep": 1,
    "completed": false,
    "progress": 16.7
}
```

### 3. 获取案例列表

```
GET /api/cases?interactivityLevel=interactive

Response:
{
    "cases": [...],
    "pagination": {...}
}
```

## 数据模型

### 互动步骤结构

```javascript
{
    stepNumber: Number,        // 步骤序号
    instruction: String,       // 学习任务说明
    expectedInput: String,     // 期望答案
    hints: [String],          // 提示数组
    feedback: String          // 反馈信息
}
```

### 案例交互等级

- `reading` - 阅读式：传统案例学习，无互动
- `guided` - 引导式：有步骤但无答题要求
- `interactive` - 交互式：必须答题推进，有反馈

## 使用流程

### 学员使用流程

1. 浏览案例库，选择感兴趣的交互式案例
2. 阅读案例背景和目标
3. 点击"开始互动学习"
4. 按步骤完成学习任务
    - 阅读任务说明
    - 思考答案（可查看提示）
    - 提交答案
    - 查看反馈和解释
5. 完成所有步骤后获得学习证书

### 教师创建流程

1. 准备真实或教学用测井案例
2. 设计学习步骤和知识点
3. 编写每个步骤的说明、答案、提示、反馈
4. 在系统中创建交互式案例
5. 发布案例供学员学习

## 教学设计理念

### 1. 基于问题学习（PBL）

- 以真实测井解释问题为导向
- 学员在解决问题的过程中学习
- 强调知识的应用而非记忆

### 2. 支架式教学

- 提供渐进式提示（scaffolding）
- 随着学习深入逐步减少支持
- 培养独立解决问题的能力

### 3. 即时反馈

- 行为主义学习理论的应用
- 正确答案得到正强化
- 错误答案得到引导和纠正

### 4. 多感官学习

- 结合测井曲线图像
- 文字说明和公式计算
- 逻辑推理和判断

## 技术实现

### 后端

- **模型**: `models/Case.js` - interactiveSteps 数组
- **控制器**: `controllers/caseController.js`
    - `startInteractiveCase()` - 开始互动学习
    - `submitInteractiveAnswer()` - 提交答案
- **路由**: `routes/cases.js` - 互动学习路由
