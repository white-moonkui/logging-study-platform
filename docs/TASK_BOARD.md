### 📝 任务详情

#### T001: 学习成就徽章体系UI ⏳

**描述**: 在原型中增加学习成就徽章展示界面

**包含内容**:

- [ ] 徽章展示区（个人主页或侧边栏）
- [ ] 徽章类型（6种）
    - 在线学习时长（铜→银→金→铂金）
    - 章节成绩（铜→银→金→钻石）
    - 问答贡献（初级→中级→高级→专家）
    - 资料上传（铜→银→金）
    - 考试通过（及格→良好→优秀→卓越）
    - 连续学习（7天→14天→30天→100天）
- [ ] 徽章详情弹窗（获取条件、当前进度）
- [ ] 徽章排行榜

**展示位置**: 学员个人主页/侧边栏

---

#### T002: 用户角色权限区分UI ⏳

**描述**: 在原型中展示不同角色的界面差异

**角色类型**:

- [ ] 管理员 - 全部功能
- [ ] 教师 - 知识库编辑/考试管理/案例管理
- [ ] 学员 - 学习/测试/查看评估

**展示方式**:

- [ ] 角色切换器（原型演示用）
- [ ] 各角色侧边栏对比
- [ ] 各角色功能权限矩阵

#### T003: 后端API展示模拟 ⏳

- [ ] 错误提示UI

**展示位置**: 页面底部或控制台面板
