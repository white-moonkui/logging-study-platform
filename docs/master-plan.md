- **文件存储**: GridFS (小文件) + 本地文件系统 (大文件)
- **认证**: JWT
- **测试**: Jest + Supertest

### 前端

- **框架**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **UI库**: Font Awesome 6.4.0
- **图表**: Chart.js 4.4.0
- **PDF阅读**: PDF.js
- **视频播放**: HTML5 Video API
- **Office预览**: Microsoft Office Online

### 工具

- **版本控制**: Git
- **包管理**: npm
- **构建**: 原生 (无需构建工具)

## ⚠️ 关键依赖与风险

### 外部依赖

1. **视频源**: 需要从外网下载哔哩哔哩视频
    - **风险**: 版权问题
    - **解决方案**: 使用内部录制的培训视频

2. **Office Online**: 需要外网访问
    - **风险**: 内网无法访问
    - **解决方案**: 使用WPS在线预览或转为PDF

3. **PDF.js**: CDN加载
    - **风险**: 内网无法访问CDN
    - **解决方案**: 本地部署PDF.js库

### 技术风险

1. **大文件上传**: 视频文件可能很大
    - **解决方案**: 分片上传 + 断点续传

2. **视频转码**: 不同浏览器支持不同格式
    - **解决方案**: 统一使用MP4 (H.264)

3. **存储空间**: 视频文件占用空间大
    - **解决方案**: 存储配额管理 + 定期清理
